# Mini clone (plain HTML/CSS/JS)

This is a tiny client-only demo of the same core workflow:

- Employee submits a request
- Manager approves/rejects
- HR approves/rejects (deducts balance)
- Approved leaves show on a simple calendar grid

## Run

Open `docs/crash-course/mini-app/index.html` in a browser.

## Reset demo data

The demo stores data in `localStorage`.

- Click **Reset demo data** in the app, or
- Open DevTools → Application → Local Storage and delete the key:
  - `leaveMiniApp:v1`
