Leave Request App

A comprehensive employee leave management system built with Laravel, Inertia.js, and Vue.js

Tech Stack Backend (Laravel) Frontend (Vue + Inertia) Database App Flows

What is this Application?

This is a Leave Request Management System that allows:

  • Employees to submit and track leave requests
  • Department Managers to approve/reject their team's requests
  • HR Admins to give final approval, manage policies, and create employees

Technology Stack Summary

Layer Technology Purpose
Backend Framework Laravel 11 PHP framework for routing, controllers, models, authentication
Frontend Framework Vue.js 3 Reactive UI components with Composition API
Frontend-Backend Bridge Inertia.js Connects Vue components to Laravel controllers without API
Starter Kit Laravel Breeze Provides auth scaffolding, components, and Tailwind setup
CSS Framework Tailwind CSS Utility-first CSS for styling
Build Tool Vite Fast frontend build tool and dev server
Database MySQL Relational database for storing data
