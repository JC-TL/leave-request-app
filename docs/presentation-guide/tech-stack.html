Technology Stack Deep Dive

Laravel 11 (Backend Framework)

Laravel is a PHP framework that provides the backend structure for our application.

What Laravel Provides

  • Routing: Maps URLs to controller methods
  • Controllers: Handle HTTP requests and return responses
  • Eloquent ORM: Object-Relational Mapping for database operations
  • Migrations: Version control for database schema
  • Authentication: Built-in user authentication system
  • Middleware: Filter HTTP requests (e.g., auth checks)

Example: A Simple Route

// routes/web.php
use App\Http\Controllers\EmployeeController;

// When user visits /employee/dashboard, call the dashboard method
Route::get('/employee/dashboard', [EmployeeController::class, 'dashboard'])
    ->name('employee.dashboard');

Example: A Controller Method

// app/Http/Controllers/EmployeeController.php
public function dashboard()
{
    $user = Auth::user();  // Get logged-in user
    
    $balances = $user->leaveBalances()  // Get related balances
        ->where('year', now()->year)
        ->get();

    // Return Inertia response (sends data to Vue component)
    return Inertia::render('Employee/Dashboard', [
        'balances' => $balances,
    ]);
}

Vue.js 3 (Frontend Framework)

Vue.js is a JavaScript framework for building reactive user interfaces.

Key Concepts

  • Components: Reusable UI building blocks (files ending in .vue)
  • Reactivity: UI automatically updates when data changes
  • Composition API: Modern way to organize component logic using script setup
  • Props: Data passed from parent to child components
  • Computed: Derived values that auto-update

Vue Component Structure

<script setup>
// JavaScript logic goes here
import { ref, computed } from 'vue';

// Define props (data from parent/server)
const props = defineProps({
    balances: Array,
});

// Reactive variable
const showModal = ref(false);

// Computed property (auto-updates when balances change)
const totalDays = computed(() => {
    return props.balances.reduce((sum, b) => sum + b.balance, 0);
});
</script>

<template>
    <!-- HTML template goes here -->
    <div>
        <h1>Total Leave Days: {{ totalDays }}</h1>
        
        <!-- Loop through balances -->
        <div v-for="balance in balances" :key="balance.id">
            {{ balance.leave_type }}: {{ balance.balance }} days
        </div>
    </div>
</template>

Frontend: Submitting Forms

import { useForm } from '@inertiajs/vue3';

// Create a form object
const form = useForm({
    leave_type: '',
    start_date: '',
    end_date: '',
    reason: '',
});

// Submit the form
function submit() {
    form.post(route('employee.store-request'), {
        onSuccess: () => form.reset(),
    });
}

Laravel Breeze (Starter Kit)

Laravel Breeze is an official starter kit that provides:

  • Authentication scaffolding (login, logout, registration)
  • Pre-built Vue components (buttons, inputs, modals, dropdowns)
  • Tailwind CSS configuration
  • Inertia.js setup
  • Layout components (AuthenticatedLayout, GuestLayout)
