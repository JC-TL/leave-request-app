Templates & Directives

Template Basics

Templates are the HTML part of Vue components. They use special syntax to make HTML dynamic.

Text Interpolation: {{ }}

<!-- Double curly braces output JavaScript values -->
<p>Hello, {{ user.name }}</p>
<p>You have {{ balance.balance - balance.used }} days left</p>
<p>Today is {{ new Date().toLocaleDateString() }}</p>
JavaScript Expressions: You can put any JavaScript expression inside {{ }}, but not statements like if/for.

1. v-if / v-else - Conditional Rendering

Show or hide elements based on conditions.

<!-- Simple condition -->
<div v-if="hasBalance">
    Show the leave request form
</div>
<div v-else>
    <p>You have no leave balance.</p>
</div>

Real Example: Employee Dashboard

<!-- Show form only if user has balance -->
<div v-if="hasBalance" class="bg-white p-6 shadow">
    <form>...leave request form...</form>
</div>

<div v-else class="bg-yellow-50 p-4">
    <p>You do not have any available leave balance.</p>
</div>

v-else-if for Multiple Conditions

<!-- Status badge with different colors -->
<span v-if="status === 'pending'" class="bg-yellow-100">
    Pending
</span>
<span v-else-if="status === 'hr_approved'" class="bg-green-100">
    Approved
</span>
<span v-else class="bg-red-100">
    Rejected
</span>

Real Example: AuthenticatedLayout

<!-- Only show HR links if user is HR admin -->
<template v-if="isHR">
    <NavLink :href="route('hr.policies')">Policies</NavLink>
    <NavLink :href="route('hr.employees')">Employees</NavLink>
</template>

2. v-for - Looping

Render a list of items by looping through an array.

<!-- Basic loop -->
<div v-for="item in items" :key="item.id">
    {{ item.name }}
</div>
Always use :key - Vue needs a unique identifier to track each item efficiently. Usually use the item's id.

Real Example: Leave Balances

<!-- Loop through balance cards -->
<div class="grid grid-cols-4">
    <div
        v-for="balance in balances"
        :key="balance.id"
        :class="['rounded-lg border p-4', getBalanceColor(balance)]"
    >
        <h4>{{ balance.leave_type }}</h4>
        <p>{{ getAvailableBalance(balance) }} / {{ balance.balance }}</p>
    </div>
</div>

Real Example: Requests Table

<tbody>
    <!-- Loop through paginated requests -->
    <tr v-for="request in requests.data" :key="request.id">
        <td>{{ request.leave_type }}</td>
        <td>{{ formatDate(request.start_date) }}</td>
        <td>{{ request.number_of_days }}</td>
    </tr>
    
    <!-- Show message if no requests -->
    <tr v-if="requests.data.length === 0">
        <td colspan="6">No leave requests found.</td>
    </tr>
</tbody>

Loop with Index

<li v-for="(item, index) in items" :key="item.id">
    {{ index + 1 }}. {{ item.name }}
</li>

3. v-model - Two-Way Binding

v-model syncs form inputs with JavaScript variables.

<!-- Input changes â†’ variable updates â†’ input shows new value -->
<input v-model="form.email" type="email" />

<!-- This is equivalent to: -->
<input 
    :value="form.email"
    @input="form.email = $event.target.value"
/>

Real Example: Login Form

<!-- Login.vue -->
const form = useForm({
    email: '',
    password: '',
    remember: false,
});

<!-- Template -->
<input type="email" v-model="form.email" />
<input type="password" v-model="form.password" />
<input type="checkbox" v-model="form.remember" />

v-model with Different Inputs

Input Type v-model Binds To
text, email, passwordString value
checkboxBoolean (single) or Array (multiple)
radioSelected option value
selectSelected option value
textareaString value

Real Example: Select Dropdown

<select v-model="form.leave_type">
    <option value="">Select Leave Type</option>
    <option 
        v-for="balance in availableBalances" 
        :key="balance.id" 
        :value="balance.leave_type"
    >
        {{ balance.leave_type }} ({{ getAvailableBalance(balance) }} days)
    </option>
</select>

4. @event - Event Handling

Use @ to listen to DOM events.

<!-- @click = v-on:click -->
<button @click="doSomething">Click Me</button>

<!-- Call function with argument -->
<button @click="cancelRequest(request.id)">Cancel</button>

<!-- Inline expression -->
<button @click="count++">Add</button>
<button @click="showModal = true">Open</button>

Common Events

Event Triggered When
@clickElement is clicked
@submitForm is submitted
@inputInput value changes
@changeInput loses focus after change
@keyupKey is released

Event Modifiers

<!-- .prevent = e.preventDefault() -->
<form @submit.prevent="submit">
    <!-- Form won't reload the page -->
</form>

<!-- .stop = e.stopPropagation() -->
<button @click.stop="doThis">Click</button>

5. :attr - Dynamic Attributes

Use : to bind JavaScript values to HTML attributes.

<!-- :href = v-bind:href -->
<a :href="url">Link</a>

<!-- Dynamic src -->
<img :src="user.avatar" />

<!-- Dynamic disabled -->
<button :disabled="form.processing">Submit</button>

<!-- Dynamic min date -->
<input type="date" :min="today" />

Dynamic Classes

<!-- Object syntax: class applies if value is truthy -->
<div :class="{ 'bg-green-50': isActive, 'bg-red-50': isError }">

<!-- Array syntax: combine static and dynamic -->
<div :class="['rounded-lg p-4', getBalanceColor(balance)]">

<!-- Ternary expression -->
<div :class="isActive ? 'text-green-500' : 'text-gray-500'">

Real Example: Mobile Menu Toggle

<!-- AuthenticatedLayout.vue -->
<div :class="{ 
    block: showingNavigationDropdown, 
    hidden: !showingNavigationDropdown 
}">
    ...mobile menu content...
</div>

Real Example: Low Balance Warning

<!-- Manager/Dashboard.vue -->
<tr
    v-for="member in teamMembers"
    :key="member.id"
    :class="{ 'bg-yellow-50': isLowBalance(member) }"
>
    <!-- Row gets yellow background if balance is low -->
</tr>
