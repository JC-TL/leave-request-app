What is Vue.js?

Vue.js is a JavaScript framework for building user interfaces. It makes your HTML dynamic and reactive.

Traditional HTML vs Vue.js

Traditional HTML Vue.js
Static content Dynamic, reactive content
Requires JavaScript for interactivity Built-in reactivity system
Manual DOM updates Automatic DOM updates
No component reuse Reusable components

Vue 3 Composition API

Your app uses Vue 3 with the Composition API, which is the modern way to write Vue components.

The <script setup> Syntax

All your components use <script setup>, which is a cleaner way to write component logic:

<!-- This is the modern Vue 3 way -->
<script setup>
import { ref, computed } from 'vue';

// Your code goes here - it's cleaner!
const count = ref(0);
</script>

<!-- vs the older Options API way -->
<script>
export default {
    data() {
        return { count: 0 };
    }
}
</script>
Why script setup? It's more concise, has better TypeScript support, and everything you define is automatically available in the template.

Real Example: PrimaryButton.vue

This is one of the simplest components in your app:

<!-- resources/js/Components/PrimaryButton.vue -->

<!-- No script needed - this is a "dumb" component -->

<template>
    <button
        class="inline-flex items-center rounded-md border 
               border-transparent bg-gray-800 px-4 py-2 
               text-xs font-semibold uppercase tracking-widest 
               text-white transition duration-150 ease-in-out 
               hover:bg-gray-700 focus:bg-gray-700"
    >
        <!-- slot = where child content goes -->
        <slot />
    </button>
</template>

Using the Component

<!-- In another component -->
import PrimaryButton from '@/Components/PrimaryButton.vue';

<!-- In the template -->
<PrimaryButton>Click Me</PrimaryButton>
<!-- "Click Me" goes into the <slot /> -->
